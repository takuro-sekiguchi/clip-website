### positionとz-indexの関係

- z-indexを有効にするためには「position: relative;」を指定する必要がある。
-  z-index は position プロパティと一緒に使う必要がある点に注意。
- 「position: absolute;」を使うと、その要素のためのスペースが確保されなくなり、他の要素の配置に影響を与えない。
